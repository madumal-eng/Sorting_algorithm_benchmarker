# Sorting Algorithms Benchmark in Python

This project compares the performance of multiple sorting algorithms
using randomly generated arrays and visualizes their execution time.

## Algorithms Implemented

- Bubble Sort
- Optimized Bubble Sort
- Selection Sort
- Swap-based Selection Sort
- Insertion Sort
- Optimized Insertion Sort
- Python Built-in Sort (Timsort)

## Features

- Random dataset generation
- Time benchmarking
- Performance visualization using Matplotlib

## Technologies

- Python
- Matplotlib

## How to Run

python3 SortingAlgos.py
